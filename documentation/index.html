Plataforma de Rifas Online

Esta es una plataforma completa de rifas online, lista para producción. Incluye todas las funcionalidades necesarias para operar rifas digitales de forma segura y profesional.

Características principales

  • Venta de tickets en línea
  • Pagos electrónicos integrados
  • Panel de administración
  • Gestión de premios y ganadores
  • Estadísticas e ingresos

¿Para quién está diseñada?

  • Emprendedores
  • Organizadores de rifas
  • Negocios digitales
  • Proyectos SaaS

Tecnologías utilizadas

Frontend (client/)

  • Next.js
  • Tailwind CSS
  • Zustand

Backend (server/)

  • Node.js
  • Express
  • TypeORM
  • MySQL
