Panel de Administración

El panel de administración permite gestionar todas las funcionalidades clave del sistema.

Funcionalidades

  • Crear, editar y eliminar rifas, proveedores y premios
  • Gestionar premios
  • Visualizar tickets vendidos y reservados
  • Graficas y analiticas de las rifas
  • Seleccionar ganadores

Reglas importantes

  • No se puede eliminar una rifa con tickets vendidos
  • No se puede eliminar un premio si tiene ganador o si está asociado a una rifa con tickets vendidos
  • No se puede eliminar un proveedor con premios activos o si está asociado a una rifa con tickets vendidos

Estas reglas garantizan la integridad de los datos y la correcta operación del sistema.
