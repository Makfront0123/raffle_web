Preguntas Frecuentes

¿Necesito GitHub?

No. El sistema se entrega como archivo ZIP / RAR listo para usar.

¿Puedo cambiar colores y logo?

Sí. Todo el frontend es completamente personalizable usando Tailwind CSS.

¿Es seguro?

Sí. El sistema incluye:

  • Autenticación JWT almacenada en cookies
  • Validaciones en backend
  • Protección de reglas de negocio

¿Puedo usarlo comercialmente?

Sí. La licencia permite uso comercial.
