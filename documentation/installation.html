Instalación

Requisitos

  • Node.js 18 o superior
  • Base de datos MySQL
  • Servidor local o VPS

Contenido del archivo ZIP / RAR

web_rifa.zip
├── client/
├── server/
├── documentation/

Instalación Backend

  1. Entrar a la carpeta server
  2. Instalar dependencias:
pnpm install

Configurar variables de entorno (.env)

DB_HOST=
DB_PORT=
DB_USER=
DB_PASSWORD=
DB_NAME=
JWT_SECRET=
NODE_ENV=
PORT=4000
JWT_SECRET=
JWT_REFRESH_SECRET=
WOMPI_PUBLIC_KEY=
WOMPI_PRIVATE_KEY=
WOMPI_INTEGRITY_SECRET=
TWILIO_ACCOUNT_SID=
TWILIO_AUTH_TOKEN=

Ejecutar el servidor

pnpm run dev

Ejecutar test del backend

pnpm test

Instalación Frontend

  1. Entrar a la carpeta client
  2. Instalar dependencias:
pnpm install

Configurar variables de entorno (.env.local)

NEXT_PUBLIC_API_URL=http://localhost:3000
NEXT_PUBLIC_WOMPI_PUBLIC_KEY=
NEXT_PUBLIC_GOOGLE_CLIENT_ID

Ejecutar el frontend

pnpm run dev

Ejecutar test del frontend

pnpm test
